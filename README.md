# ⏱️ Time Tracker

A simple and responsive web application for tracking work hours and estimating weekly earnings. Designed for employees who want to keep an accurate record of their worked hours and calculate their expected paycheck after estimated taxes.

---

## 🚀 Features

* Clock In / Clock Out system
* Automatic work time calculation
* Daily worked hours summary
* Total accumulated hours
* Weekly payment estimation
* Gross pay calculation
* Estimated tax deductions (Federal + New York)
* Net pay estimation
* Local data persistence using Local Storage
* Reset weekly data
* Mobile-friendly responsive interface

---

## 💰 Payment Calculation

Current hourly rate:

**$25.00/hour**

Estimated deductions include:

* FICA
* Federal Tax
* New York State Tax
* NY SDI / Paid Family Leave

The application automatically calculates:

* Gross Pay
* Total Deductions
* Estimated Net Pay

## 🔒 Data Storage

No backend or database is required.

All information is stored locally using the browser's **Local Storage**, allowing users to close the browser and continue where they left off.
